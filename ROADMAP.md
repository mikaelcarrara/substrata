# Roadmap

Substrata's strategic milestones for 2026.

## 2026 Plan

- **Q1 2026**: Developer Experience Foundation
    - **Official CLI**: A robust command-line tool for scaffolding new Substrata projects, validating configuration files, and running build pipelines.
    - **JSON Schema**: Formal schema definitions for `tokens.json` to enable IDE validation, autocompletion, and error highlighting.

- **Q2 2026**: Design-to-Code Pipeline
    - **Figma Sync**: Bidirectional synchronization to push token updates from Figma variables directly to the code repository, ensuring design intent is preserved.
    - **VS Code Extension**: IDE integration providing intelligent autocompletion, inline color swatches, hover documentation, and linting for token usage.

- **Q3 2026**: AI Agents SDK (Substrata Intelligence)
    - **MCP Server**: A Model Context Protocol server exposing the token graph to AI assistants (Cursor, Windsurf, Claude), allowing them to understand and suggest correct semantic tokens.
    - **Semantic Search**: Vector-based search allowing developers to query tokens using natural language (e.g., "standard padding") instead of memorizing variable names.

- **Q4 2026**: Autonomous Operations
    - **Auto-Refactor Agent**: An autonomous CLI agent that scans legacy codebases, identifies hardcoded values (hex, px), and intelligently maps them to the nearest Substrata tokens.
    - **Generative Theming**: AI tool to generate complete, accessible theme configurations (colors, typography) from text prompts or image mood boards.
